# Digital Clock

A responsive Digital Clock web application built using HTML, CSS, and JavaScript that displays the current time and date in real-time with a clean, modern, and user-friendly interface.

## Features

- Real-time digital clock
- Displays current date and time
- Automatic time updates
- Responsive and modern user interface
- Easy to use

## Technologies Used

- HTML
- CSS
- JavaScript

## Live Demo

https://madhuri-yelure.github.io/Digital-Clock/

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
